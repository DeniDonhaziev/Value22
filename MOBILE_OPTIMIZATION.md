# Мобильная оптимизация Value Marketplace

## Обзор

Этот документ описывает все улучшения, внесенные для оптимизации веб-приложения Value Marketplace под мобильные устройства.

## Основные улучшения

### 1. Адаптивная верстка

#### CSS Grid и Flexbox
- Использование CSS Grid для адаптивных сеток
- Flexbox для гибких макетов
- Автоматическое изменение количества колонок в зависимости от размера экрана

#### Медиа-запросы
```css
/* Мобильные устройства */
@media (max-width: 768px) { ... }

/* Планшеты */
@media (min-width: 769px) and (max-width: 1024px) { ... }

/* Десктопы */
@media (min-width: 1025px) { ... }
```

### 2. Типографика

#### Адаптивные размеры шрифтов
- Базовый размер: 16px (предотвращает зум на iOS)
- Адаптивные заголовки: от 1.75rem до 5rem
- Улучшенная читаемость на маленьких экранах

#### Улучшенная типографика
```css
body {
  font-size: 16px;
  line-height: 1.6;
  -webkit-text-size-adjust: 100%;
  -ms-text-size-adjust: 100%;
}
```

### 3. Touch-friendly интерфейс

#### Минимальные размеры touch targets
- Кнопки: минимум 48x48px
- Ссылки: минимум 44x44px
- Инпуты: минимум 48px высота

#### Улучшенные состояния
```css
.btn, button {
  min-height: 48px;
  padding: 12px 16px;
  touch-action: manipulation;
  -webkit-tap-highlight-color: transparent;
}
```

### 4. Навигация

#### Мобильное меню
- Гамбургер-меню для мобильных устройств
- Полноэкранное мобильное меню
- Плавные анимации открытия/закрытия
- Легкий доступ к основным функциям

#### Адаптивная навигация
- Скрытие элементов на маленьких экранах
- Перегруппировка элементов навигации
- Оптимизация для touch взаимодействий

### 5. Формы и инпуты

#### Предотвращение зума на iOS
```css
input, textarea, select {
  font-size: 16px; /* Предотвращает зум на iOS */
  min-height: 48px;
  padding: 12px 16px;
}
```

#### Улучшенные состояния фокуса
- Четкие индикаторы фокуса
- Улучшенная доступность
- Поддержка клавиатурной навигации

### 6. Изображения и медиа

#### Адаптивные изображения
```css
img {
  max-width: 100%;
  height: auto;
  border-radius: 12px;
}
```

#### Ленивая загрузка
- Intersection Observer API
- Fallback для старых браузеров
- Оптимизация производительности

### 7. Производительность

#### Оптимизация анимаций
- Отключение анимаций на устройствах с низкой производительностью
- Поддержка `prefers-reduced-motion`
- Оптимизация для 60fps

#### Оптимизация скролла
```css
body {
  -webkit-overflow-scrolling: touch;
  scroll-behavior: smooth;
}
```

### 8. Доступность

#### ARIA атрибуты
- Правильная разметка для скринридеров
- Поддержка клавиатурной навигации
- Улучшенная семантика

#### Контрастность
- Соответствие WCAG 2.1 AA
- Поддержка высокого контраста
- Адаптация под темную тему
