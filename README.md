# Value Marketplace

Современная платформа для малого бизнеса - аналог Авито с фокусом на магазины и предпринимателей.

## 🚀 Быстрый запуск

### 1. Установка зависимостей
```bash
npm run install-all
```

### 2. Запуск приложения
```bash
npm run dev
```

Приложение будет доступно по адресу: **http://localhost:3000**

## 📋 Возможности

### Для покупателей:
- 🔍 Поиск магазинов и товаров
- 📱 Адаптивный дизайн
- ⭐ Рейтинги и отзывы
- 💬 Встроенный чат с продавцами
- 📍 Геолокация магазинов

### Для продавцов:
- 🏪 Создание витрины магазина
- 📦 Управление товарами
- 📊 Статистика продаж
- 💰 Система платежей
- 📈 Продвижение магазина

## 🛠 Технологии

### Backend:
- **Node.js** + **Express.js**
- **SQLite** (разработка) / **Firebase Firestore** (продакшн)
- **JWT** аутентификация
- **Multer** для загрузки файлов
- **Twilio** для SMS уведомлений

### Frontend:
- **React** + **TypeScript**
- **Tailwind CSS** для стилизации
- **React Router** для навигации
- **Axios** для HTTP запросов
- **Lucide React** для иконок

## 🔧 Настройка Firebase (опционально)

Для использования Firebase вместо SQLite:

1. Следуйте инструкциям в файле `FIREBASE_SETUP.md`
2. Создайте файл `.env` в папке `server`
3. Добавьте переменные окружения Firebase

## 📱 Основные страницы

- **Главная** (`/`) - Лендинг с поиском и категориями
- **Магазины** (`/shops`) - Список всех магазинов
- **Товары** (`/products`) - Каталог товаров
- **Регистрация** (`/register`) - Создание аккаунта
- **Вход** (`/login`) - Авторизация
- **Профиль** (`/profile`) - Личный кабинет
- **Мой магазин** (`/my-shop`) - Управление магазином

## 🚀 Развертывание

### Локальная разработка:
```bash
npm run dev          # Запуск в режиме разработки
npm run build        # Сборка для продакшна
```

### Продакшн:
1. Настройте Firebase
2. Добавьте переменные окружения
3. Соберите клиент: `npm run build`
4. Запустите сервер: `npm start`

## 📞 Поддержка

Если у вас возникли вопросы или проблемы:

1. Проверьте консоль браузера на ошибки
2. Убедитесь, что все зависимости установлены
3. Проверьте, что порты 3000 и 5000 свободны
